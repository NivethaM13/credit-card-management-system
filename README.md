# Credit Card Management System

A full stack Credit Card Management System built using:

- React JS
- FastAPI
- MySQL
- SQLAlchemy
- JWT Authentication

## Features

- User Registration
- User Login
- Add Credit Card
- View Saved Cards
- Delete Card
- Make Payment
- Transaction History

## Technologies Used

Frontend:
- React JS

Backend:
- FastAPI
- Python

Database:
- MySQL
